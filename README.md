# Tejas Porwal Website

Static personal website for Tejas Porwal.

## Overview

This project is a lightweight multi-page website built with plain HTML, CSS, and a small amount of vanilla JavaScript.

Current pages:

- `index.html`: main landing page
- `work.html`: selected work index
- `project-national-roadmap.html`: strategy case study
- `project-compiler-optimizations.html`: engineering case study
- `project-recursion-academy.html`: founder case study
- `index-print.html`: print-oriented legacy draft

## Notes

- The homepage hero uses a lightweight animated node graph and rotating role copy.
- Shared work and detail-page styling lives in `pages.css`.
- The site is designed to be easy to edit directly before any future migration to a framework.
